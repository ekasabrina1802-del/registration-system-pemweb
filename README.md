# 🎓 Registration System - Pemweb

A web-based student registration system developed for the Web Programming course using PHP, MySQL, Apache (XAMPP), HTML, and CSS.

## 📌 Features

- Student registration form
- Input validation
- Session-based error handling
- MySQL database integration
- Unique registration number generation
- Printable registration receipt
- Responsive and modern user interface

## 🛠️ Technologies Used

- PHP
- MySQL
- Apache
- HTML
- CSS

## 📂 Project Structure

```bash
registration-system-pemweb/
│
├── pendaftaran.php
├── hasil_pendaftaran.php
├── koneksi.php
├── db_pendaftaran_mahasiswa.sql
└── README.md
```

## ⚙️ Installation

### 1. Clone repository

```bash
git clone https://github.com/ekasabrina1802-del/registration-system-pemweb.git
```

### 2. Move project to htdocs

```bash
C:/xampp/htdocs/
```

### 3. Start XAMPP

Run:

- Apache
- MySQL

### 4. Import database

- Open http://localhost/phpmyadmin
- Create database:

```sql
db_pendaftaran_mahasiswa
```

- Import file:

```bash
db_pendaftaran_mahasiswa.sql
```

### 5. Run project

Open browser:

```bash
http://localhost/PraktikumFramework
```
